# Prompts séance 08

Ce dossier regroupe les prompts extraits de la page `index.html`.

Ils servent à animer l'atelier même si aucun candidat volontaire ne propose spontanément un livrable.

## Ordre conseillé

1. `01-prompt-maitre-structurer-livrable.md`
2. `02-prompt-nettoyage-contenu.md`
3. `03-prompt-presentation-orale.md`
4. `04-prompt-feedback.md`
5. `05-scenario-secours-sans-volontaire.md`

## Mode d'utilisation rapide

1. Choisir un livrable candidat réel ou utiliser le scénario de secours.
2. Remplir les zones entre crochets.
3. Coller le prompt dans l'outil IA de l'atelier.
4. Lire la réponse avec le groupe.
5. Demander aux participants ce qu'ils gardent, corrigent ou vérifient.

## Rappel important

Les réponses générées doivent toujours être relues, vérifiées et adaptées au contexte réel de l'atelier.
