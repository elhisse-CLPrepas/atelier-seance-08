# Scénario de secours - aucun candidat volontaire

Utiliser ce scénario si aucun candidat ne propose de livrable pendant l'atelier.

## Objectif

Créer une situation de démonstration réaliste pour exécuter les prompts avec le groupe.

## Livrable fictif à renseigner

```text
Titre du projet :
Fiche d'aide pour préparer un entretien avec l'IA

Public cible :
Candidats débutants qui veulent mieux préparer leur présentation orale.

Besoin traité :
Beaucoup de candidats ont des idées, mais ils ont du mal à les organiser clairement pour les présenter en 3 à 5 minutes.

Version actuelle du travail :
J'ai préparé une fiche avec quelques conseils :
- écrire son objectif ;
- demander à l'IA de reformuler ;
- préparer une introduction ;
- noter les questions difficiles ;
- s'entraîner à parler.

Le document est encore trop général. Il manque une structure claire, des exemples concrets, une checklist et une méthode pour vérifier que le candidat ne récite pas seulement une réponse produite par l'IA.
```

## Déroulé d'animation

1. Coller le livrable fictif dans `01-prompt-maitre-structurer-livrable.md`.
2. Lire la structure proposée par l'IA.
3. Demander au groupe : qu'est-ce qui est utile ? qu'est-ce qui manque ? qu'est-ce qui doit être vérifié ?
4. Coller le résultat dans `02-prompt-nettoyage-contenu.md`.
5. Utiliser `03-prompt-presentation-orale.md` pour générer une mini-présentation.
6. Terminer avec `04-prompt-feedback.md` pour produire les questions de retour.

## Variante rapide

Si le temps est court, exécuter seulement :

1. le prompt maître ;
2. le prompt feedback ;
3. une discussion collective de 5 minutes.
